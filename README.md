## npm包使用说明
- styled-components
> 管理css样式
- react-transition-group
> react动画效果
- redux
> redux包管理数据
- react-redux
> 为react专属的redux，提供provider等组件
- immutable
> 防止数据被意外修改,将state改变为immutable对象
- redux-thunk
> 在action里面写函数
- axios
> ajax获取数据
- react-router-dom
> 使得react支持路由


## 特殊说明
- PureComponent
> 使用PureComponent组件，可以提升性能，但是最好配合immutable.js一起使用
